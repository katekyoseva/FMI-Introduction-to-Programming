# Задачи

Под създаване на масив с `n` (`a <= n <= b`) броя елементи разбираме предварителното инициализиране на масив с `b` на брой елементи, в който ще въвеждаме и ще работим с `n` брой числа. С други думи реалният размер на масива ще е `b`, а размера, с който работим е `n`.

**Задача 1**

Напишете програма, която приема от конзолата естествено число `n` (`1 <= n <= 32`), попълва масив от `n` на брой цели числа, и прилага върху всеки от елементите функция (напишете я) за повдигане на втора степен. Принтирайте променения масив.

Вход:
```
5 
2 4 5 1 9
```

Изход:
```
4 16 25 1 81
```

**Задача 2**

Напишете програма, която приема от конзолата естествено число `n` (`1 <= n <= 32`). Принтирайте сумата на уникалните цифри в числото (тези които се срещат само веднъж).

Вход:
```
349135792
```

Изход:
```
19
```

**Задача 3**

Напишете програма, която приема `3` естествени числа (размери на `3` масива от цели числа, като тези размери са между `1 <= n <= 10`), попълваме трите масива от конзолата и ги записваме в нов масив в следния ред:

**1 число от първия, 1 число от втория, 1 число от третия** и повтаряме като пропускаме масивите, които са се изчерпали.

**Забележка**: Попълваме масивите един по един.

Принтирайте получения масив.

Вход:
```
3 1 2 3
5 10 20 30 40 50
2 100 200
```

Изход:
```
1 10 100 2 20 200 3 30 40 50
```

**Задача 4**

Напишете програма, която чете от конзолата естествено число `n` (`4 <= n <= 32`), създава два масива от `n` на брой цели числа, които се попълват от конзолата и проверява дали първия масив е пермутация на втория. Може да използвате само един допълнителен помощен масив.

Вход:
```
4
1 2 3 4
3 1 2 4
```

Изход:
```
true
```
---
Вход:
```
3
5 9 2
2 3 6
```

Изход:
```
false
```

**Задача 5**

Инверсия в една пермутация от числа имаме, когато числа `a, b` с индекси `i, j` са подредени `i < j` и `a > b`. С други думи `а` да е на по-начална позиция в пермутацията от `b` и `a > b`.

Напишете програма, която приема от конзолата естествено число `n` (`1 <= n <= 32`), попълва масива с `n` на брой цели числа (прочетени от конзолата) и принтира броя инверсии в масива.

Вход:
```
4 
7 2 5 4
```

Изход:
```
4
```
---
Вход:
```
3
1 2 3
```

Изход:
```
0
```

**Задача 6**

Напишете програма, която прочита символен низ от конзолата (`до 10 символа`) (отварящи и затварящи скоби `(`, `)`) и принтира дали изразът от скоби е валиден.

Вход:
```
((()))
```

Изход:
```
true
```
---
Вход:
```
))()
```

Изход:
```
false
```

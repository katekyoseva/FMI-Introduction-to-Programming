**Задача 1**

Напишете програма, която принтира масив от числата от 1 до 10, използвайки указатели (не може да използвате индексиране `[i]`).

**Задача 2**

Напишете функция, която получава два указателя към `char` (char* input, char* result) и размера на тези масиви. В `result` копирайте  масивът `input` като на мястото на всяка цифра поставете интервал.

Вход:
```
This1Is2A3Message.
```

Изход:
```
This Is A Message.
```

**Задача 3**

Напишете функция, която симулира оператор `+=`. Функцията трябва да бъде тип `void` и да манипулира директно подадените параметри.
Използвайте функцията с въведени от конзолата числа.

Примерно извикване на функцията:
```cpp
int a = 5;
int b = 3;

operatorPlusEquals(a, b);

std::cout << a;
```

Изход:
```
8
```

**Задача 5**

Напишете функция, която приема указател към int (int* ptr) и двумерен масив от цели числа (прочетен от конзолата). Намерете реда от двумерния масив с най-голямо средно аритметично от всички редове и пренасочете ptr към този ред. Принтирайте намерения ред извън функцията, използвайки подадения ѝ указател.

Вход:
```
3 3
1 2 3
4 5 6
7 8 9
```

Изход:
```

```

# Задачи

**Задача 1**

Напишете рекурсивна функция, която по подадено число `n` пресмята и връща факториела му.

Вход:
```
5
```

Изход:
```
120
```

**Задача 2**

Напишете рекурсивна функция, която по подадени цели числа `n` и `k`, да пресмята `n`, повдигнато на `k-та` степен.

Вход:
```
3 3
```

Изход:
```
27
```

**Задача 3**

Напишете рекурсивна функция, която да пресмята `n`-тото число от редицата на Фибоначи.

Вход:
```
5
```

Изход:
```
3
```

Пояснение:
Редицата изглежда по този начин: 0 1 1 2 3

**Задача 4**


Изход:
```

```

**Задача 5**

Напишете рекурсивна функция, която при подадено цяло число `n` изписва на конзолата всяко число от `1` до `n`, като на всеки нов ред броят на числата се намалява докато не стигне `1`. С други думи имате следния примерен вход: :D

Вход:
```
4
```

Изход:
```
1 2 3 4
1 2 3
1 2 
1
```

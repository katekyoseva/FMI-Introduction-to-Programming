# Задачи

**Задача 1**

Напишете функция, която приема параметри две числа и връща резултата от първото число повдигнато на степен второто. Принтирайте резултата.

Вход:
```
2 3
```

Изход:
```
8
```

**Задача 2**

Напишете функция, която не приема параметри и принтира на конзолата ```"Hello, world!"```.

**Задача 3**

Напишете функции ```isUppercase``` и ```isLowercase```, които приемат параметър символ и връщат съответно дали е главна или малка буква. Приемаме, че работим само с латински букви. Принтирайте резултата.

Вход (за isUppercase):
```
j
```

Изход:
```
false
```

Вход (за isLowercase):
```
l
```

Изход:
```
true
```

**Задача 4**

Вход:
```
4 5 *
```

Изход:
```
20
```

Изход:

**Задача 8**

**Задача 9**

Напишете програма, която приема две числа и принтира на конзолота резултата от целочисленото им делене и деленето с остатък. Разделете програмата си на отделни функции!

Вход:
```
16 3
```

Изход:
```
5 1/3 
```

**(С други думи "5 и една трета")**

**Задача 10**

Напишете програма, която приема цели числа `x` и `a`, `b`, `c`, `d`, `e`, `f` (граници на затворени интервали - `[a, b]`, `[c, d]`, `[e, f]`) и определя дали точката е в обединението на първите 2 и не е в третия. Принтирайте резултата. Валидирайте интервалите. Помислете какви функции са ви нужни за решението на задачата.

Вход:
```
3 2 5 4 9 -15 2
```

Изход:
```
true
```
